Ovaj privatni repozitorij predstavlja drugi repozitorij putem kojeg ćete predavati rješenja vaših uradaka za postavljene
zadatke. U mapi `ZADACI` nalaze se tekstovi zadataka. Ova grupa zadataka je u biti samo jedan zadataka koji obuhvaća Command Pattern.


Rješenje zadatka (**da ovaj put samo jedan** :-) treba se nalaziti unutar mape `RJESENJA`. Upute za rad s Gitom u sprezi s GitHubom nalaze se na
sustavu Merlin kao i popratne vježbe. Preporuka je da prvo ponovite taj dio, jer smo sve u vezi rada s GitHubom prošli
na vježbama (u okviru OOP).

&nbsp;

![wrkflow](https://img.shields.io/badge/solutions-workflow-yellow?style=for-the-badge)

Vaš tijek rada (eng. workflow) **ovaj put treba biti malo drugačiji**:

1. Prvi commit treba biti u glavnu granu nakon što ste napravili GUI s tablicom, ali bez padajućih izbornika
2. Kreirajte novu granu `MenuBarFeatures` i drugi commit treba biti u tu granu nakon što ste kreirali padajuće menije (funkcionalne) &Rarr; kada je to gotovo napravite **pull request** na koji ću vam dati povratnu informaciju može li se napraviti spajanje `merge` u glavnu granu &rarr; što ću i napraviti ako je do tada prihvatljivo rješenje
3. Za realizaciju `Tool Bar` kreirat ćete novu granu `ToolBarFeatures` za koju ponavljamo raniji postupak (pull request + merge)
4. Četvrtki commit u novu granu `Pop-up Menu Features` za rezultat treba imati funkcionalni `Pop-up Menu` nakon čega slijedi `pull request`+ `mwrge` &rarr; ovdje bi rješenje u biti bilo gotvo u potpunosti što znači da pripadno spajanje u glavnu granu predstavlja gotovo rješenje
